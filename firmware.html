Für den Einsteiger ist die Auswahl des passenden Gerätes schwierig. Faustformeln sind:

  • Lieber dual-band (2,4 und 5GHz) als single band, denn ein Band ausschalten kann man immer noch und sie kosten kaum mehr Geld.
  • MIMO und insbesondere MU-MIMO Geräte können gleichzeit eintreffende Signale anhand deren Richtung auseinanderhalten - dafür brauchen sie die vielen Antennen.
  • Wenn die Installation nicht einfach ist, dann die Installation beim nächsten Freifunk Treffen vorführen lassen.
  • Mindestens 8MB Flashspeicher und 64MB RAM - so köller auch neueste Versionen der Firmware im vollen Umfang genutzt werden

  • Eine schnelle CPU für mehr als 15Mbit/s - das Tunneln aus dem lokalen Netzwerk zu unseren Servern geht dann schneller

  • Wer keinen Nerv hat, eine unserer Helfer-Firmen kontaktieren.

Firmware Erstellung    

Beim Bauen der Firmware werden die Commit-Ids des gluon Repositories und des Site-Konfigurations-Repositories in der Firmware verankert. Diese Ids kann man in der Startseite der Erweiterten Einstellungen im Config Mode ersehen. Die Git Id ist genau der Commit auf dem die Firmware gebaut wurde. Falls dort ein Plus (+) am Ende erscheint ist dies der "Dirty"-Flag. Die Id wurde erzeugt mit 

git describe --always --dirty=+
